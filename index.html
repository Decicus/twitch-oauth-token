Twitch OAuth Token

This is an OAuth token generator for the Twitch API. It includes the scopes for both the "New Twitch API" (aka Helix) and the "old" API called Kraken or V5.
The source code for this page is available on GitHub

Scope reference:

    Scopes with an underscore (_) are tied to Kraken / V5.
    Scopes with a colon (:) belong to the Helix / new Twitch API.
    The scope openid includes neither, but belongs to Kraken / V5.
