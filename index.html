(Unofficial) Twitch OAuth Token Generator

This is an unofficial OAuth token generator for the Twitch API. It includes the scopes for both the "New Twitch API" (aka Helix) and the "old" API called Kraken / V5

Source code on GitHub

Scope reference

    Scopes with an underscore (_) are tied to Kraken / V5.
    Scopes with a colon (:) belong to the Helix / new Twitch API.
    The scope openid is a small exception, but belongs to Kraken / V5.
